## Modo de uso

Configurar parámetros del server en el archivo `serverConfig.json`. El mismo debe ubicarse en la misma carpeta del ejecutable (.exe).

## Formato de `serverConfig.json`
```
{
  "host":"localhost",
  "port":8080,
  "serve_path":"" 
}
```
Serve path debe contener el path completo a la carpeta con los recursos a servir.
